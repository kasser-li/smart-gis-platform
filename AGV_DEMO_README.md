# 🤖 AGV 智能调度演示模块

## 功能概述

在原有 GIS 平台基础上新增的 AGV（自动导引车）智能调度演示模块，用于展示 AGV 车队管理、路径规划、任务调度等核心功能。

## 主要功能

### 1. 📍 AGV 实时监控
- **车队列表**：显示所有 AGV 的编号、状态、电量、负载、位置
- **状态标签**：工作中（蓝色）、空闲（绿色）、充电中（橙色）、离线（灰色）、故障（红色）
- **电量监控**：实时显示电量百分比，低电量（<20%）自动告警
- **位置追踪**：在地图上实时显示 AGV 位置

### 2. 🗺️ 路径规划
- **可视化路径**：在地图上绘制 AGV 行驶路线
- **起点/终点标记**：绿色为起点，红色为终点
- **工作站管理**：预设 8 个工作站（原料仓库、生产线、成品仓库、质检区等）

### 3. 📋 任务管理
- **任务类型**：搬运、分拣、充电、巡检
- **任务状态**：等待、进行中、已完成、已取消
- **AGV 分配**：支持自动分配或手动指定 AGV
- **优先级设置**：低/中/高三个优先级

### 4. 🔋 充电站管理
- **充电站位置**：地图上显示 3 个充电站
- **状态监控**：显示充电站占用/空闲状态
- **自动充电**：AGV 电量<15% 时自动前往充电

### 5. 🎮 模拟演示
- **开始/暂停**：启动或暂停 AGV 运动模拟
- **重置**：恢复初始状态
- **自动行为**：
  - AGV 移动时电量逐渐消耗
  - 低电量自动前往充电
  - 充电完成后自动恢复空闲状态

### 6. 📊 运营看板
- **在线 AGV 数量**
- **进行中任务数**
- **平均电量**
- **今日完成任务数**

## 技术实现

### 前端技术栈
- Vue 3 + TypeScript
- Element Plus UI 组件库
- Leaflet 地图引擎
- Vite 构建工具

### 核心组件
- `AGVDemo.vue` - AGV 演示主页面
- 路由：`/agv`

### 数据模拟
- 5 台 AGV 车辆
- 3 个充电站
- 8 个工作站/路径点
- 实时任务队列

## 使用方法

### 启动开发服务器
```bash
cd /root/.openclaw/workspace/smart-gis-platform/frontend
npm run dev
```

### 访问页面
- 主页（GIS 地图）：http://localhost:5173/
- AGV 演示：http://localhost:5173/agv

### 从主页跳转
在 GIS 主页顶部工具栏点击「🚗 AGV 演示」按钮即可跳转。
